Экранированный символ пригоден только для вывода на экран, он не может быть частью оператора. например, ##59$ то экранированный символ ";" в режиме dosurq и ripurq не будет считаться комментарием можно использовать например ##44$ (экранированный символ ",") чтобы вывести строку с запятой в операторе btn #/$ - заменяется на перенос строки #%/$ - заменяется на перенос строки #$ - заменяется на пробел #%$ - заменяется на пробел Системные переменные: urq_mode - флаг, указывающий на особенности в проигрывании данного квеста (подробности ниже) нужен для проигрывания старых квестов, написанных под другие интерпретаторы count_локация - счетчик заходов на локацию, меняется от операторов btn, proc, goto music - если присвоить этой переменной файл с музыкой, то она начнёт проигрываться и будет повторяться по окончанию common - изменение этой переменной приведёт к тому, что оператор btn станет вызывать локацию не :common а :common_{значение} при переходах current_loc - название текущей локации. Меняется только при btn переходах previous_loc - название предыдущей локации. Меняется только при btn переходах style_dos_textcolor - число от 0 до 15. Меняет цвет текста на дос-цвета. style_textcolor - меняет текста на указанный (в десятичном формате или шестнатиричном с префиксом #) style_backcolor - меняет цвет фона (в десятичном формате или шестнатиричном с префиксом #) Пример: style_backcolor = "#000" /* цвет фона теперь черный */ image - выводит указанную картинку на экран inv_[предмет] - возвращает количество [предмета] в инвентаре. Если присвоить этой переменной числовое значение, то количество предметов в инвентаре станет равным этому значению. Пример: inv_Яблоко = 4 if inv_Яблоко > 0 then pln у меня есть как минимум одно яблоко! [предмет] - возвращает количество [предмета] в инвентаре Пример: inv+ 4, Яблоко if яблоко then pln у меня есть как минимум одно яблоко! Комментарии: /* комментарий */ Операторы: :[метка] метка, на которую можно перейти с помощью операторов goto, proc, btn if [условие] then [тогда] {else [иначе]} Пример: if Яблок > 4 then pln у меня много яблок! else pln У меня мало яблок music [файл] проигрывает указанный файл один раз, останавливает предыдущий играющий файл вызванный через music play [файл] проигрывает указанный файл один раз cls принудительно очищает экран от текста и кнопок proc [метка] переход на указанную локацию и выполнение квеста до оператора end, после чего возврат на изначальную позицию (увеличит счетчик заходов на локацию, переменная count_[метка]) forget_proc "забыть" о переходе через proc, возврата при встрече end не будет end оператор окончания выполнения.
